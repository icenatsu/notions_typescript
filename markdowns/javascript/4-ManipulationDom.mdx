## Itérer des HTML collection ou Node

<Pre>
```js 
// Boucle for (HTML collection ou Node)
const cards = document.getElementsByClassName("card");
for (let i = 0; i < cards.length; i++) {
  console.log("Card:", cards[i]);
}

// Boucle for of (HTML collection ou Node)
const titleCards = document.querySelectorAll(".card-title");
for (let titleCard of titleCards) {
  console.log("titleCard", titleCard);
}

// forEach 
  // Requiert une conversion en array pour une HTML collection
Array.from(cards).forEach((card) => {
  console.log("Card", card);
});
  // Pas de conversion en array pour une Node Collection
titleCards.forEach((titleCard) => {
  console.log("titleCard", titleCard);
});
```
</Pre>

## Manipulation de la structure

###  appendChild: Ajouter un élément à la fin du noeud parent

<Pre>
```js 
const p = document.createElement("p");
p.textContent = "AJouter un élément à la fin du noeud parent"
p.style.color = "white"
const container = document.getElementById("container")
container.appendChild(p)
```
</Pre>

![console](/appendChild.webp "appendChild")


###  removeChild: Supprimer un élément à la fin du noeud parent

<Pre>
```js 
container.removeChild(p)
```
</Pre>

![console](/removeChild.webp "removeChild")

###  replaceChild: Remplacer un enfant par un autre

<Pre>
```js 
container.appendChild(p)
const p2 = document.createElement("p")
p2.textContent = "Je remplace le paragraph p par p2"
p2.style.color = "white"

container.replaceChild(p2, p)
```
</Pre>

![console](/replaceChild.webp "replaceChild")

###  insertBefore: Insérer un élément avant un autre élément

<Pre>
```js 
container.insertBefore(p, p2)
```
</Pre>

![console](/insertBefore.webp "insertBefore")

```html
  </div>

```
</Pre>

<Pre>
